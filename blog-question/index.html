1. Difference among map, filter, forEach and find.

map filter forEach find
map() is used for getting all single value from an array. filter() give that values from an array which is fulfilled the specific condition. forEach() seperates all values from array but not return values. find() return first value when first condition fulfilled.

3. What is event loop? How does it work?

Event loop : Event loop is a cycling ordering loop with execution of call stack. Event loop perform step by step execution.
How does work : Firstly add new operation with call stack, then call to API, and then data will sent to event queue after timeout, finally complete execution of event loop.
